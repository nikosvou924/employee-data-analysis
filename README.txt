Project Description

This project is a Workforce Analytics System developed in Python.
It processes employee data for a company using CSV files, JSON files, and a web API.

The dataset contains employee information such as:
- id
- fullname
- gender
- department
- salary
- bonus

The system supports:
- loading and sorting employee records
- input validation
- counting and grouping records by department
- calculating salary statistics
- object-oriented processing of employee data
- searching and filtering employee records
- data analysis with pandas
- data visualization with matplotlib

The data used in this project was generated with Mockaroo.
The purpose of the project is to demonstrate file handling, data structures,
object-oriented programming, and basic data analytics in Python.
